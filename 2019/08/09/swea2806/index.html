  • 백트래킹 : 해를 찾는 도중에 막히면 되돌아가서 다시 해를 찾아가는 기법 / DFS에 Pruning을 적용한 것 / 가도 되지 않는 루트는 고려하지 않는다. / 일종의 완전 탐색 기법
  • N-Queen : 크기가 N*N인 체스판 위에 퀸 N개를 서로 공격할 수 없게 놓는 것


  • 절차 :
  1. DFS 수행
  2. 유망한 노드 검토 : 방문한 노드를 포함해서 유망한 노드이면 서브트리로 이동, 그렇지 않으면 백트래킹 수행
  3. 서브 트리 이동 : 방문한 노드의 하위 노드로 이동, 재귀를 통해 DFS 수행
  4. 백트래킹 : 방문한 노드를 가지치기, 상위 노드로 백트래킹, 다시 DFS 수행

이와 별도로 대각선 검사하는 코드도 한번 볼 필요가 있는것 같다.
또한, board[] 배열은 인덱스는 행, 인덱스가 지칭하는 원소는 해당 행의 열을 의미한다.
